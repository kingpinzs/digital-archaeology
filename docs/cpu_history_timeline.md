# Historical CPU Timeline (1971 - ~2004)

A comprehensive list of CPUs from early microprocessors through the Pentium 4 era, including major rivals/competitors.

## Early Microprocessors (1971-1978)

| Year | Intel | Rivals | Notes |
|------|-------|--------|-------|
| 1971 | **4004** (4-bit, 740kHz) | - | First commercial microprocessor |
| 1972 | **8008** (8-bit, 0.5MHz) | - | First 8-bit microprocessor |
| 1974 | **8080** (8-bit, 2MHz) | Motorola 6800 | Basis for CP/M systems |
| 1975 | - | MOS 6502, Zilog Z80 | 6502 used in Apple II, C64; Z80 in TRS-80 |
| 1978 | **8085** (8-bit, 5MHz) | - | Enhanced 8080 |

## 16-bit Era (1978-1985)

| Year | Intel | Rivals | Notes |
|------|-------|--------|-------|
| 1978 | **8086** (16-bit, 5-10MHz) | - | Foundation of x86 architecture |
| 1979 | **8088** (8-bit bus) | Motorola 68000, Zilog Z8000 | Used in IBM PC |
| 1982 | **80186/80188** | - | Integrated peripherals |
| 1982 | **80286** (16MHz) | Motorola 68010 | Protected mode introduced |

## 32-bit Era (1985-1995)

| Year | Intel | Rivals | Notes |
|------|-------|--------|-------|
| 1985 | **80386 (i386)** (12-40MHz) | Motorola 68020 | First 32-bit x86, virtual memory |
| 1989 | **80486 (i486)** (25-100MHz) | Motorola 68040, AMD Am486 | Integrated FPU and cache |
| 1992 | - | Cyrix 486SLC/DLC | Budget alternatives |

## Pentium Era (1993-2000)

| Year | Intel | Rivals | Notes |
|------|-------|--------|-------|
| 1993 | **Pentium (P5)** (60-200MHz) | AMD K5, Cyrix 6x86 | Superscalar architecture |
| 1995 | **Pentium Pro (P6)** | - | Out-of-order execution |
| 1997 | **Pentium MMX** | AMD K6, Cyrix 6x86MX | SIMD instructions |
| 1997 | **Pentium II** | AMD K6-2, Rise mP6 | Slot 1 package |
| 1999 | **Pentium III** | AMD K6-III, AMD Athlon | SSE instructions |

## Pentium 4 Era (2000-2004)

| Year | Intel | Rivals | Notes |
|------|-------|--------|-------|
| 2000 | **Pentium 4 (Willamette)** | AMD Athlon/Thunderbird | NetBurst architecture |
| 2002 | **Pentium 4 (Northwood)** | AMD Athlon XP | 0.13μm process |
| 2004 | **Pentium 4 (Prescott)** | AMD Athlon 64 | 90nm, high power consumption |

## Cloning Recommendations

### Best for Learning (Simpler)
- **Z80** or **6502** - Well documented, still used in retro computing
- **8086** - Foundation of x86, manageable complexity

### Sweet Spot (Challenging but Achievable)
- **80386** - First true 32-bit x86, paging/protected mode
- **68000** - Clean CISC design, used in early Macs/Amigas

### Advanced (Very Complex)
- **486** - Includes FPU, cache, more pipeline stages
- **Pentium (P5)** - Superscalar, but still understandable

## Key Architecture Families

### Intel x86 Lineage
```
4004 → 8008 → 8080 → 8086 → 286 → 386 → 486 → Pentium → P4
```

### Motorola 68k Lineage
```
6800 → 68000 → 68010 → 68020 → 68030 → 68040 → 68060
```

### MOS/WDC 65xx Lineage
```
6502 → 65C02 → 65C816 (16-bit)
```

### Zilog Lineage
```
Z80 → Z180 → Z280 → Z380
```
